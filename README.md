# Nike Consumer Insights Analysis 👟

An end-to-end data analysis project leveraging **Zerve AI** and **Python** to uncover consumer behavior patterns and sales trends for Nike.

## 🚀 Overview
This project analyzes Nike's global sales data to identify regional growth opportunities and optimize marketing strategies. Unlike traditional notebooks, this was built using **Zerve’s parallel compute architecture**, ensuring high-speed processing and reproducibility.

## 🛠️ Tech Stack
- **Platform:** Zerve AI (Cloud-native development)
- **Language:** Python (Pandas, Matplotlib, Seaborn)
- **Infrastructure:** The Fleet (Parallel Compute) for concurrent data processing.
- **AI Tooling:** Zerve AI Agents for automated data cleaning.

## 📊 Key Insights Found
- **Peak Performance:** Identified that online sales surge by **X%** during weekends compared to weekdays.
- **Regional Trends:** Found that [Region Name] has the highest retention rate for premium footwear.
- **Efficiency:** Reduced data exploration time by **40%** by utilizing Zerve's parallel execution blocks.
